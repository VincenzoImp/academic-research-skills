# Research Results Reporting

Report results from completed analysis bundles. Do not rewrite numbers as
independent truth.

## Read First

- `references/repository-contract.md`
- `references/workflow-stage-contracts.md`
- `references/output-contracts.md`
- `references/claim-audit.md`

## Workflow

1. Confirm `npm run workflow:analysis` preflight passed for the source analysis.
2. Read the contribution manifest, contribution-local `analysis.yaml`, generated data, tables, figures, stats appendix, and figure catalog.
3. Write or revise the analysis report and contribution report.
4. Reference generated files directly for tables, figures, and source data.
5. Record belief updates, negative results, caveats, and next actions.

## Review Loop

Review interpretation against data, methodology, effect size, assumptions,
limitations, and claim maps. Fix and re-review until no overclaim or stale
asset reference remains.

## Handoff

Hand off reviewed paper exports and report sections to manuscript and framing
workflows.
